# ✅ NoteSnap — Gen AI Academy APAC Submission Checklist

Complete these steps in order to fully submit your project.

---

## 🔑 Step 1: Set Up Your API Key

- [ ] Go to https://aistudio.google.com/apikey
- [ ] Sign in with your Google account
- [ ] Click "Create API Key" → copy it (starts with `AIza...`)
- [ ] Keep it safe — treat it like a password, never share it publicly

---

## 🧪 Step 2: Test Your App Locally

- [ ] Open `index.html` in Google Chrome
- [ ] Paste your API key in Step 1 box
- [ ] Upload a real PDF (e.g., any NCERT chapter or your own notes)
- [ ] Try all 5 summary modes and confirm they work
- [ ] Take a screenshot of the working app with a summary visible
- [ ] Take a second screenshot showing the Exam Prep mode

> **Tip**: The app works best with text-based PDFs. Scanned image PDFs may not extract text.

---

## 🔧 Troubleshooting

| Problem | Solution |
|---|---|
| "API Error: 400" | Double-check your API key — no extra spaces |
| "No text extracted" | Your PDF is image-scanned — try a different PDF |
| App not loading | Make sure you're using Chrome or Firefox (not Safari) |
| Summary seems wrong | Try adding a subject hint (Physics, Chemistry, etc.) |
| Key rejected | Regenerate key at aistudio.google.com/apikey |
